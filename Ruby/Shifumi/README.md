# Shifumi en ruby
Un Shifumi en ruby utilisant gosu pour les graphismes.

#Utilisation
Utiliser la commande :
> [ruby] Main.rb

Vous devrez avoir installé ruby (2.x) et gosu sur votre ordinateur pour faire fonctionner l'application.
Si vous être sur Windows vous pourrez aussi faire ceci :

> Main.exe

A noter que cette commande peut aussi être replacée par une execution normal de l'executable (double clic).
